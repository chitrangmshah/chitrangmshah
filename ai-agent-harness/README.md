# ai-agent-harness

Notes on **AI agent harnesses** — the scaffolding around a language model that turns it from a text generator into an agent that can act.

A *harness* is the machinery an AI, LLM, or SLM runs inside: the tools it can call, the context files it reads (like `AGENTS.md`), its memory, and the loop that lets it plan, act, and observe. This folder collects learnings, patterns, and references on building and working with such harnesses.
